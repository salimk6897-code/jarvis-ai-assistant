# 🤖 JARVIS AI Assistant

Hey there! I'm Salim Malik, and this is one of my favorite projects 
I've built — a personal AI assistant inspired by Iron Man's Jarvis!

I built this completely from scratch using Python and Google's 
Gemini AI API. It took me a lot of learning, debugging, and 
late night coding sessions — but honestly, every minute was worth it! 

## What can Jarvis do?

Jarvis isn't just a simple chatbot. He actually remembers your 
conversation, has a personality, and can help you with real tasks!

- 💬 **Chat naturally** — just talk to him like a real assistant
- 🧠 **Memory** — he remembers everything you tell him in the session
- 🧮 **Calculations** — ask him to calculate anything step by step
- 📝 **Writing** — need an email or letter? Just ask!
- 😄 **Jokes** — he tells surprisingly good tech jokes
- 💡 **Advice** — sometimes you just need a fresh perspective
- 🎭 **Personality** — he calls you Boss and has Iron Man's wit!

## What I used to build this

- Python 3.14
- Google Gemini API
- Prompt Engineering techniques (zero-shot, few-shot, chain of thought)

## Want to run it yourself?

It's pretty simple actually!

1. First install the library:
   pip install google-genai

2. Get a free API key from aistudio.google.com

3. Add your API key to the code

4. Run it:
   python jarvis2.py

## Commands

Once you run it, you can say things like:
- "calculate 20% of 5000"
- "write an email to my manager"
- "tell me a joke"
- "give me some advice"
- "bye" to exit
